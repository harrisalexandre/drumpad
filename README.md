# 🥁 DrumPad

Um Drum Pad simples e divertido desenvolvido com HTML, CSS e JavaScript puro.

## 🎵 Demonstração

Acesse o projeto:

👉 https://harrisalexandre.github.io/drumpad/

## ✨ Funcionalidades

* Reprodução instantânea de efeitos sonoros
* Interface simples e responsiva
* Controle por clique
* Controle por teclado
* Funciona diretamente no navegador
* Sem dependências externas

## 🎹 Controles

| Tecla  | Som      |
| ------ | -------- |
| Espaço | FAHH     |
| W      | WOW      |
| A      | AMONG US |

Também é possível reproduzir os sons clicando nos botões.

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📂 Estrutura do Projeto

```text
drumpad/
├── index.html
├── fah.mp3
├── wow.mp3
├── among-us.mp3
└── README.md
```

## 💻 Executando Localmente

Clone o repositório:

```bash
git clone https://github.com/harrisalexandre/drumpad.git
```

Acesse a pasta:

```bash
cd drumpad
```

Abra o arquivo `index.html` em seu navegador.
